# Hostel Management System

A simple hostel management system in Java.

## How to Install and Run this Project

### Prerequisites

*   Make sure you have Java Development Kit (JDK) installed on your system.

### Compilation

1.  Open your terminal or command prompt.
2.  Navigate to the `hostel` directory of the project.
3.  Compile the Java source files using the following command:

    ```bash
    javac src/*.java
    ```

### Running the Application

1.  After successful compilation, run the application from the `hostel` directory using the following command:

    ```bash
    java -cp src Main
    ```
